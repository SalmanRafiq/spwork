In this plugin you can identify which plugins are required for your wordpress website.
 
== Description ==
 
This plugin provide you the plugin name and download option link for your website.
 
== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Done and Ready.
